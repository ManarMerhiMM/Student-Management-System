# 🎓 Student Management System

A software project designed to manage and organize student-related data efficiently.

---

## 📌 Overview

The **Student Management System** is a structured web application that demonstrates:

- CRUD operations (Create, Read, Update, Delete)  
- Data organization and record handling  
- Basic system design for educational management  
- User interaction through a structured interface  


---

## 🚀 Features

### 📂 Student Records Management
- Add new student information  
- View all stored students  
- Update existing student details  
- Delete student records  

### 🔍 Search & Retrieval
- Search for specific students  
- Display detailed information  

### 📊 Academic Handling
- Store course-related data  
- Manage student performance

### 🧠 Core Concepts
- Database for storing records  
- Logical flow for CRUD systems

## ▶️ How to Run

### 1. Clone the repository
```bash
git clone https://github.com/ManarMerhiMM/Student-Management-System.git
```

### 2. Navigate into the project
```bash
cd Student-Management-System
```

### 3. Install dependencies
```bash
composer install
```
### 4. Start Laravel's Server
```bash
php artisan serve
```

### 5. Host a mySQL database locally

### 6. Configure `.env` file for database connection details
#### Example:
```bash 
DB_CONNECTION=mysql
DB_PORT=3306
DB_DATABASE=student_management
DB_USERNAME=root
DB_PASSWORD=
```
### 7. Run the mySQL database server

### 8. Start browsing
---

## 🎯 How It Works

1. User interacts with the system interface  
2. Inputs student data (name, ID, etc.)  
3. System stores data in database  
4. User can:
   - Retrieve records  
   - Modify existing entries  
   - Remove records  

This follows a standard **CRUD-based system design**.
